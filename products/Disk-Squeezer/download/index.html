最新版-2.0版本

  • 大小：20.9 MB
  • 2024年11月30日发布
  • 界面几乎重写，使用相对位置，适配各个屏幕
  • 首次加入了动画
  • 美化界面，使界面更加好看
  • 再次加入多语言，支持简体中文，繁體中文，English
  • 首次加入了系统托盘
  • 开始榨干时，窗口会自动隐藏
  • 当榨干结束时，将使用 Windows 消息提示
  • 程序文件大小变小，许多无用的组件被删除

特殊版-CMD1.0版本

  • 解压后大小：2.16 MB
  • 2024年8月19日发布
  • 英文名：Disk-Squeezer-Console
  • 首个可以通过调用命令直接榨干的版本
  • 使用之前，需要在其所在目录打开终端或者将终端目录引导到其所在目录，输入命令调用；或者将其放在PATH变量中或System32中，隐蔽性好，还可再任何目录直接用命令调用
  • 在终端中输入squeeze /h获取帮助

2.0-Beta版本

  • 解压后大小：87.4 MB
  • 2024年8月1日发布
  • 2.0-Alpha版本中首次加入了图形化界面，但及其粗糙，本版本为Alpha版完善许多功能而来
  • 首次使用Qt库，界面比Alpha版好看了不少
  • 写入模式首次不使用主线程
  • 命令模式代码几乎重写，比2.0-Alpha快了不止几十个量级

1.5版本

  • 大小：174 KB
  • 2024年2月25日发布
  • 注意：该版本打包时没有打包 Visual C++ Runtime(VCRuntime) 运行库，需要自己安装，安装方法请访问 Internet
  • 由32位程序转为64位程序
  • 首次支持多语言，支持简体中文和English双语言切换
  • 增加“关于”选项
